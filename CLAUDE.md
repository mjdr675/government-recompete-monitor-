# Government Recompete Monitor

## What this project does
Monitors government contract recompete opportunities, focusing on janitorial/facilities contracts. Identifies upcoming contract expirations and competitive leads from federal procurement data.

## Key files
- `janitorial_recompete_report.py` — main script that processes procurement data and generates recompete reports
- `janitorial_recompete_report.csv` — raw recompete report output
- `janitorial_recompete_report_clean.csv` — cleaned/processed version of the report
- `tier_a_competitive_leads.csv` — high-priority competitive leads filtered from the report
- `requirements.txt` — Python dependencies

## How to run
```bash
pip install -r requirements.txt
python janitorial_recompete_report.py
```

## Dependencies
See `requirements.txt`. Install with `pip install -r requirements.txt`.

## Data
- Input: Federal procurement CSV data
- Output: Recompete opportunity reports filtered by contract type, value, and expiration timeline
- `tier_a_competitive_leads.csv` contains the highest-priority leads

## Conventions
- Python script handles all data processing
- CSV outputs are regenerated by running the script
- Keep raw and clean CSVs separate
